Excel Cleaner

A simple tool to allow you to upload an XLSX excel file and have certain columns wiped, and others replaced with an SHA512 key, constructed from selected fields within the file.

NOTE:the entire process happens in memory, with no files being stored in any temporary file store. This is to ensure that no private data is accidentally left on the server should there be an error. This also means that there are size limitations placed on the files being uploaded.

If you don't like uploading these files to remote servers, there's also a python script that can do the same anonymisation locally.

This is running on glitch so you can review the running code for this at a anytime here.

View the README on the underlying project for more information on the format. A sample config file and a sample data file are also provided as an examples.
